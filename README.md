## Project Overview

A website that provides users with a marketplace and a forum to discuss and buy/rent movies. Trending movies are displayed on the home page and an individualized movie description with the trailer link, movie duration, rating, and movie synopsis are displayed for each movie. In addition the forum allows users to comment and write posts about their favorite movies.
